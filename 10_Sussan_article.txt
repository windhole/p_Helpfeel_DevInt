# 2025.03.06：Helpfeel 社員インタビュー 鈴木孝宏さん


## タイトル
// 北海道からフルリモート勤務で環境の良さを満喫しながら、プロフェッショナル達と働ける醍醐味



## リード

Helpfeel開発部の新しいメンバーを紹介するインタビューシリーズ「New Developers Interview」。第10回は、鈴木 孝宏さん（@）です。

鈴木さんは、
// メディアキャプチャーツール[Gyazo](https://gyazo.com/)のプロダクトエンジニアとして、GyazoのMac向け・iOS向けプラットフォームの開発に携わっています。また、バックエンドや課金機能なども担当しています。札幌からフルリモートで勤務しています。

インタビュアーは、Helpfeel CTOの秋山さん。


## 見出し1（これまでのキャリア、入社の経緯）

```
フリーランス時代に業務委託
今後のことを相談したら
「正社員でどうですか？」
```

秋山さん：Helpfeelに入社するまでのキャリアを教えてもらえますか？　大学では、未踏に参加していたんですよね。

鈴木さん：はい、修士のときですね。

秋山さん：何がきっかけで未踏に応募したんですか？

鈴木さん：僕の指導教員の美馬義亮先生（※1）が未踏のPM経験者で、ご自身も未踏を経験された方でした。その美馬先生から「やってみないか」と言われまして。

※1：美馬 義亮 – 公立はこだて未来大学 -Future University Hakodate-
https://www.fun.ac.jp/faculty/mima-yoshiaki
- Yoshiaki Mima / Future University Hakodate
  https://www.fun.ac.jp/~mima/j/index.html


秋山さん：お、そうなんですね。

鈴木さん：美馬先生経由で、増井先生（現・Helpfeel社フェロー）のことも知りました。

秋山さん：美馬先生も、増井先生と近い領域ですもんね。未踏では、どんなテーマに取り組んだんですか？

鈴木さん：スマートフォンのテキスト入力です。スマートフォンらしいインタラクションでテキスト入力をもっと効率よく簡単にしてみようという研究でした（※2）。時間帯とか場所といったコンテキスト情報を元に文章を組み立てようという。

※2：未踏IT人材発掘・育成事業：2013年度採択プロジェクト概要（鈴木孝宏PJ）
https://www.ipa.go.jp/jinzai/mitou/it/2013/gaiyou_i-3.html

秋山さん：面白いですね。最近の日本語入力システムだと、画面に出ているテキストを元に挙動を変える機能がありましたが、時間や場所などスマートフォンならではのコンテキスト情報を変換に利用するのも、確かにアリですね。

鈴木さん：結局、なかなか思ったような成果にはならなかったんですけど、結構チャレンジングなことをしてたなという感じですね。

秋山さん：その後は、どんなキャリアを進んできたんですか？

鈴木さん：2015年からは、モバイルアプリのエンジニアをしていました。初めはAndroidをよく見ていたんですが、2018年ぐらいからiOSをメインで見るようになりました。その後フリーランスになって、iOSだけに絞ってビジネスをしていたという感じです。

秋山さん：そこは、モバイルアプリの会社？

鈴木さん：いえ、Webメディアの会社でした。その中の新規事業みたいな小さな部署で、スマホアプリの開発をしていました。

秋山さん：じゃあ、周りにもアプリ開発ができる人がいたんですね。

鈴木さん：それが、そうでもなくて。Web開発がメインの人が多くて、モバイルアプリは社内で勉強しながらという感じでした。

秋山さん：フリーランス時代は、どんな感じだったんですか？　Helpfeelも最初は業務委託でお願いしていましたが、他にも何社かあったんですよね。

鈴木さん：メインがHelpfeelで、他にスポットで定期的に頂くお仕事が2、3社ありました。フリーランスは、忙しさに波がありまして。調子が良いときはいいんですが、逆に仕事を受けすぎたりして……。

秋山さん：フリーランスは、そうなりがちですよね。

鈴木さん：「仕事ありますか」っていろんな人に声をかけていると、依頼があったときに断り切れなくて……。そうやってどんどん仕事を受けちゃって、働きすぎて調子を崩すこともありました。

秋山さん：フリーランスの大変さも一通り経験して、今に至るんですね。

鈴木さん：ホントに仕事がない時期もありましたし。

秋山さん：Helpfeelで、業務委託として関わるようになったきっかけは？

鈴木さん：そのときは、まさに仕事がない時期でした。Helpfeelでデザイナーとして働いている山田さんが大学時代の知り合いで「うちで募集してるよ」と声をかけてくれたんです。

秋山さん：業務委託でジョインする前は、Helpfeelのイメージってありましたか？

鈴木さん：そうですね……どんなイメージだったんだろう？　Gyazoをやっていることは知っていました。Scrapboxもちょっと遊んだけど、そんなに触っていなかったので、どちらかというとGyazoのイメージが強かったですね。

秋山さん：その後、Helpfeelの正社員になったきっかけは？

鈴木さん：話がちょっと戻りますが、フリーランスになった理由が、家族へのサポートを増やすためだったんです。それが、だんだん子どもも大きくなって、僕も仕事に集中できる時間が増えてきたというのがありました。僕自身が、フリーランスを続けていく理由が徐々に少なくなってきたんです。

そこで秋山さんに条件面を含め相談させて頂いたら「じゃあ正社員でどうですか」と提案して頂いて。

秋山さん：ハハハハ。そうでした。

鈴木さん：自分としては、結構衝撃だったんです。

秋山さん：本当ですか！

鈴木さん：いや、そうなんです。想定していた内容とは全然違っていて。最悪、切られるかなと思っていたので。

秋山さん：切るだなんて、全然思ってなかったですよ。


```
大きなことをガツンと任せてもらえるのは
プレッシャーでもあり
やる気が出るところでもあり
```

秋山さん：Helpfeelに、正社員として入ろうと思った決め手は何でしたか？

鈴木さん：正直、Helpfeelの居心地が良かったんです。これまで働いてきた会社と比べて特長があるなと思って。アカデミックな研究とビジネスを繋いでいるところとか。社内の方々が普通に論文出してたりしますし。そういうのもすごく魅力的でした。その上で、自分自身も等身大でいられるというか、片ひじ張らずにいられるというか。

秋山さん：そういう感じの会社ですよね。

鈴木さん：そんな自分らしくいられるってところが、すごく居心地が良かったんですよね。それもあって、社員っていうご提案を受けたときに、すごく安心したというか。まだここに居られるんだって。

秋山さん：なかなかエモい話ですね。

鈴木さん：本当に、すごく嬉しかったんです。

秋山さん：実際に働いてみて、業務委託と正社員の違いって何かありましたか？

鈴木さん：業務委託時代と比べると、見える範囲はかなり広がりましたよね。久しぶりに圧倒されたところでした。

会社員として守るべき社内のルールだったりとか、そういえば会社員ってこんな感じだなというのは確かにありました。ただ実際の普段の業務では、そこまで変わってない気がしています。

秋山さん：入社後に変化したところってありますか？　仕事でもプライベートでも。

鈴木さん：思った以上に働きやすい会社だというのはありました。

「ちょっと子供が風邪なんで」とか「子供のお迎えがあるんで」とか、「ちょっと会議ずらせますか」みたいなことも言いやすい環境でした。こちらとしては申し訳なく思うのですが、「家族を大事にして」とか「子供優先で」みたいに言ってくれるので、優しい会社だなというのはすごく思ったところですね。

秋山さん：仕事でのやりがいだったり、醍醐味、面白さを感じるのは、どういうときですか？

鈴木さん：結構、大き目なことも任せてもらえるところです。

例えば、業務委託で入ってすぐに、アプリをスクラッチから作り直して、リリースまで持っていくという、大きなこともさせてもらいました。

最近だと、新しい決済方法としてアプリ内課金にチャレンジさせてもらったり、大き目なことをガツンと任せてもらえるのは、プレッシャーでもあり、やる気が出るところでもあり、「何とかやってやるぜ！」と、やりがいというか、楽しいところですね。



```
プログラミングするにも
北海道って、すごくいい環境だなって
ちょっと思ってます。
```

秋山さん：一緒に働いている同僚は、どんな方が多いですか？

鈴木さん：僕の観測できる範囲だけでいくと。みんなプロフェッショナルだと思います。

秋山さん：なるほど。

鈴木さん：プロフェッショナルなのに、いろんな人の意見をちゃんと取り入れるし。なんか言葉があれですけど、おごり高ぶってないというか。

秋山さん：謙虚ってことですかね？

鈴木さん：そうですね。プロフェッショナルであり謙虚であり。意見の違いを認め合える人が多くて。そこはすごく良いと思ってます。

秋山さん：良い関係ですね。

鈴木さん：そうなんです。なんかこう、違う意見があっても思い切って言えちゃう、みたいな。それこそ、心理的安全性の高いチーム環境だなと思います。

### 秋山さん：実際に入社してからの学びと言いますか、気付きやインサイト・発見なんかはありますか。

鈴木さん：そうですね・・・ちょっと分野によるかもしれないですけど。いろんな技術を遊んでみるスタイルの人が多いなという印象があります。僕もそういうことしますけど、それ以上に新しい技術とか、もちろん枯れた技術とかもとことん遊んでいく人が多いなと入社してから気付きました。

秋山さん：なるほど。遊んでいくっていうのは面白いですね。

鈴木さん：そこから新しい機能に繋がっていったり、さらにビジネスまで繋げていけてるのはすごいと思います。


### 秋山さん：鈴木さん自身としては、こんな仕事をしたい、こんなエンジニアになってみたいという将来像ってありますか。

鈴木さん：具体的なイメージはないんですけれども、なんとなくこのままカタカタとプログラムを書いている自分が思い浮かんでいますね。

秋山さん：引き続きプログラミングを頑張っていくぞ、っていうイメージですかね。

鈴木さん：なのかなぁと思ってますね。結構流れに身を任せるタイプで、どうなっていくかは会社次第かもしれないし、世の中の状況次第かもしれないし。

何より、拠点を北海道に戻したのもあって、北海道の人たちの活動をニーズというか活動を見ながら変わっていくのかなと考えています。

秋山さん：元々ご出身が北海道なんでしたっけ。

鈴木さん：そうです。

秋山さん：やっぱり北海道には思い入れがありますか。

鈴木さん：そうですね。北海道も札幌も好きじゃないぜみたいな時期もしばらくあったんですけど。

秋山さん：ハハハ、そんな反抗期みたいな時期があったんですね。

鈴木さん：あらためて戻ってくると北海道っていい環境だなと。プログラミングの実装に集中した後、外に出た時の自然とかオンオフがはっきりできる、すごくいい環境だなって、ちょっと思ってます。

(風穴さん)：札幌は、ITコミュニティも大きいですよね。

鈴木さん：こんなに活発だったんだというのを、ここ1ヶ月で感じてました。ちょうど今週末にLTに出るんです。

秋山さん：いいですね。素晴らしい。

鈴木さん：Helpfeelのことも宣伝しておきます。

秋山さん：北海道民の人にも、うちの会社入ってもらえるとうれしいです。たしか、3人くらいいた気がする。

鈴木さん：そうなんですね。

お話すると、きっと居心地の良さがわかると思います。このプロフェッショナルの方々と一緒に自分のレベルを1段階引き上げるきっかけになると思うので、ぜひチャレンジしてほしいなというところですかね。

秋山さん：北海道道民会を作るきっかけになるといいですね。

鈴木さん：いいですね。
